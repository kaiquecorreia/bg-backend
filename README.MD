## Instalação

Clonar pasta do do projeto

Configurar um banco de dados local ou servidor remoto do mongodb

criar um arquivo .env com o seguinte conteúdo

```
DB_URI=mongodb://localhost:27017/biototal
```

Abrir a pasta do projeto pelo terminal

executar o comando

```
npm install ou yarn
```

após executar, com o banco mongodb rodando digite o comando no terminal

```
npm start ou yarn start
```

Caso queira rodar com o servidor nodemon

```
npm dev ou yarn dev
```
